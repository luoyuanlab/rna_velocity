#  **Toward precision RNA velocity in single-cell transcriptomics: challenges and perspectives**

## **Description**
To benchmark RNA velocity model performance on complex tumor microenvironment, we simulated 2,000 CD8⁺ T cells from each of the immune‑sensitive and immune‑resistant areas, focusing on PTPN11, KRAS, and MAPK1. Among the three genes across both areas, only PTPN11 in the immune-sensitive area shows non-stepwise transcription; KRAS and MAPK1 in the sensitive area, and all three genes in the resistant area, display clear stepwise changes in transcription rates, which are difficult for conventional RNA velocity models to capture.
